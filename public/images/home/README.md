# Home Hero Layered Image Names

Upload these two files to this folder to render the homepage hero right visual:

home-hero-chakra-bg.png
home-hero-subject-foreground.png

Supported extensions are: png, webp, jpg, jpeg.

Behavior:
- chakra-bg rotates continuously.
- subject-foreground is rendered on top with drop shadow depth.
- If files are missing, a built-in visual fallback appears.
